기본구조

|-- 싱글 --|
<div class="inputRange1" data-form-mode="range">
	<div class="area">
		<input type="range" title="대체텍스트" min="0" max="100" step="1" value="">
	</div>
</div>

|-- 멀티 --|
<div class="inputRange1" data-form-mode="range">
	<div class="area">
		<input type="range" title="대체텍스트" min="0" max="100" step="1" value="">
		<input type="range" title="대체텍스트" min="0" max="100" step="1" value="">
	</div>
</div>
  • min : 최소 값
  • max : 최대 값
  • step : 변경되는 값 범위

옵션(data 정의)

옵션 정의
옵션 타입 기본값 설명
data-output number null 현재 값(output) : 갯 수 만큼 노출
data-datalist number null label 숫자 지정: 갯 수 만큼 노출

활용

싱글
멀티
옵션 (현재값) : [data-output="num"]
옵션 (Label) : [data-datalist="num,num.num"]
옵션 (현재값 + Label) : [data-range-option="datalist"]
샘플) 조기은퇴진단
  • 외부영역에서 range의 값을 받아 와야 할 경우
  • 외부영역에 data-range-id="ID"값과 .inputRange1의 id="ID"값을 동일하게 적용함.
